debilnicek, notes made for web. (beta)

All it does is recording notes, and aims to do it well.


Writing anything into title / search field in header will search and filter your current notes for given text, if there is no match and enter is pressed a new note will be created.
All changes are always saved automatically as you type.


This app depends on advanced web technologies, works only with modern web browsers, read recent releases of Firefox and Chrome. As an offline web application, all you need to do is bookmark this address, when you access it even while not connected to internet it will still load with all notes that you have made. You can keep editing and making new notes. Be aware that wiping your browser history will remove them all.
